# NexTask
 
NexTask is a full-featured productivity dashboard and task management application built for tracking daily habits, one-time goals, and weekly targets. The project focuses on advanced JavaScript logic, real-time UI updates, and complete data persistence across browser sessions.
 
## Features
 
- Three task behavior types: One-Time (permanent) · Daily (midnight reset + streak tracking) · Weekly (cumulative progress toward a target)
- SVG Progress Ring animating based on overall completion percentage
- Live Metrics: Completion Momentum %, Active Day Streak, Due Today count
- Due Date intelligence: auto-labels tasks as 📅 Due Today · ⏰ Due Tomorrow · ⚠️ Overdue
- Gamified milestone ranking: 🌱 Beginner → 🚀 Achiever → 🔥 Performer → 👑 Productivity Master
- Full CRUD operations: Add, Edit (modal), Delete, Complete/Undo
- Real-time search with live card filtering
- Combined filter system: type filter × status filter
- Complete localStorage persistence surviving browser refresh
- Midnight reset detection using setInterval date comparison
- Responsive layout with mobile hamburger sidebar
